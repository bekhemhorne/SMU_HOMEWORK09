Summary: Latitude vs X

In this project we are to analyze how the weather will change in relation to the locations distance to the equator.

We used OpenWeatherMap API to assemble the dataset of over 500 cities.

Next we will use Matplotlib to plot different aspects of the weather versus the latitude. We will look at the Temperature, Wind, Humidity and Cloudiness.
